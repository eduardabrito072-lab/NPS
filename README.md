# 📊 NPS Ranking

Aplicação simples para controle de NPS por operador, com ranking automático.

## 🚀 Funcionalidades

- Lançamento de notas NPS (0 a 10)
- Cálculo automático de pontuação
- Ranking em tempo real
- Histórico de lançamentos
- Exportação como imagem
- Controle por mês

## 💾 Armazenamento

Os dados são salvos no navegador (localStorage).

## 📱 Uso

1. Abra o app
2. Vá em "Lançar"
3. Escolha operador e nota
4. Veja o ranking atualizado

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
